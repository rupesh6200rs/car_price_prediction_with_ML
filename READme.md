# Car Price Prediction using Machine Learning

## Project Overview

This project predicts the selling price of used cars using Machine Learning. Various factors such as car age, fuel type, transmission, present market price, kilometers driven, and ownership history are used to estimate the selling price.

## Technologies Used

 Python
 Pandas
 Scikit-learn
 Matplotlib
## Dataset Features

 Car_Name
 Year
 Present_Price
 Driven_kms
 Fuel_Type
 Selling_type
 Transmission
 Owner

## Target Variable

 Selling_Price

## Machine Learning Model

Linear Regression

## Evaluation Metrics

 R² Score
Mean Absolute Error (MAE)
 Root Mean Squared Error (RMSE)

## Output

The project generates:

 Predicted selling prices
 Model performance metrics
 Actual vs Predicted Price visualization
